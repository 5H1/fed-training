CSS Units Demonstration

Demonstration of five headings, each showcasing a different font-size unit — rem, em, px, vw, and vh.

Heading styled with font-size: 2rem

Uses rem — scales relative to the root font size, ensuring consistent and predictable sizing.

Heading styled with font-size: 2.5em

Uses em — scales relative to the element’s own or parent font size, which can accumulate when nested.

Heading styled with font-size: 48px

Uses px — provides fixed, absolute sizing ideal for precise layout control.

Heading styled with font-size: 7vh

Uses vh scales with viewport height — useful for hero sections using viewport-based sizing.

Heading styled with font-size: 5vw

Uses vw — scales with viewport width; ideal for large headings.
