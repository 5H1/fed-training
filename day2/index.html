INLINE CSS

Styles are applied directly to a specific HTML element using the style attribute.

INTERNAL CSS

Styles are written inside the "style" tag within the "head" section of the HTML file.

EXTERNAL CSS

Styles are kept in a separate .css file and linked to the HTML file using the tag.

SPECIFICITY

Inline styles > Internal CSS = External CSS (when comparing same selectors).
